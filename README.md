# helm-awscli-kubectl

Small Alphine Docker image with Helm, AWS CLI, and Kubectl. 

## Docker Hub
[https://hub.docker.com/r/chiangyaw/helm-awscli-kubectl](https://hub.docker.com/r/chiangyaw/helm-awscli-kubectl)

* Version: 3.19
